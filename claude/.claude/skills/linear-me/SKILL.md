## Pre-requisites
If the following are unavailable, abort the skill and explain why.

1. The linear MCP server

## Instructions
Gather context on my current linear status. You should use this to find the following in priority order.

- My current active team or teams.
- My current project (planned or started but preferring started). If one is not started, the most recent project I've been assigned to lead. Projects that are completed are not candidates for this.
- My active issues that are in progress
- My other assigned issues
- My comments on others issues or documents, particular technical specs or project descriptinos

You can use this context to add issues appropriately.

## Guidelines

- For my current project. Find this by listing projects where I am the lead, across all teams (don't restrict to the team where my recent issues live). Prefer started over planned; among planned, prefer the most recently updated. Skip completed and canceled. Do not infer the project from a recently-created or recently-updated project unless I am explicitly the lead
