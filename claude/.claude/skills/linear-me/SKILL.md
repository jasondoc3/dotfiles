name: linear-me
description: A skill to pull my current linear scope. This includes current team, project, and in progress / in review issues.
---

The linear MCP server is a pre-requisite to using this skill. It must be available. If it's not bail, and tell me why.

Gather context on my current linear status. You should use this to find the following in priority order.

- My current active team
- My current project (planned or started but preferring started)
- My active issues that are in progress
- My other assigned issues

You can use this context to add issues appropriately.

Use the following rules when creating issues unless otherwise stated. They should be:

- In TODO status
- Assigned to me
- In my current project
- In my current team
